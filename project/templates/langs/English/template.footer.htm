О сервисе UI Wallet

UI Wallet позволяет каждому пользователю оплачивать покупки и отправлять деньги, не раскрывая данных банковской карты или другой личной информации. Платежи через UI Wallet также покрываются Программой протекции, которая теперь действует 180 дней с момента совершения платежа.

  • Идентифкация
  • Помощь
  • Контакты
  • Соглашение
  • Вход
  • Регистрация
